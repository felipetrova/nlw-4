# How work's

**Install dependences**

```bash
npm
# or
yarn
```

**Run server**

```bash
npm run dev
# or
yarn dev
```

**Run tests**

```bash
npm run test
# or
yarn test
```

**Create Migration**

```bash
npm run typeorm migration:create -n NameMigration
# or
yarn typeorm migration:create -n NameMigration
```

**Run Migrations**

```bash
npm run typeorm migration:run
# or
yarn typeorm migration:run
```
